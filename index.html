Projects

M.K.G. Mixer

HTML | SASS | JavaScript | jQuery

The M.K.G Mixer lets you create your own beats by playing cut up samples of music that are mapped to each button. Buttons are key bound to keys on the keyboard but can also be played by clicking. Option to play with two different song samples.

Live Project GitHub

Bottom Bin Cinema

HTML | SASS | JavaScript | jQuery | RESTful API

Need a movie to watch tonight? We've got you covered! Just pick a genre and a decade and Bottom Bin Cinema will return a random movie to watch with a rating 5/10 or lower. Created with Connie Tsang using pair programming.

Live Project GitHub

Guess the Lyric

HTML | SASS | JavaScript | React | RESTful API

Search for your favorite karaoke song and get some practice by filling in the missing lyrics. Guess the Lyric was built with a group at Juno College and required collaboration and strong communication between all members.

Live Project GitHub
